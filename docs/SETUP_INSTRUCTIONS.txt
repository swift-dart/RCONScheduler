## Quick Start
1. **Download** PumpkinScheduler.exe
2. **Create** a `.env` file in the same folder as the executable
3. **Add** this line to your `.env` file:
   ```
   FERNET_KEY=REPLACE_WITH_YOUR_KEY
   ```
4. **Generate** your encryption key (see below)
5. **Run** PumpkinScheduler.exe
